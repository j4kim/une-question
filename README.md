# Une question

[une-question.3sdl.ch](https://une-question.3sdl.ch)

Une application qui permet de répondre à une question de manière publique. Par exemple pour planifier qui amène quoi à un pique-nique. Les utilisateurs peuvent donner leur réponse après avoir consulté celles des autres.

## Développement

```
php -S localhost:1234
```
